# Rain Time Extractor

Công cụ đọc nhiều file TXT khí tượng và xuất thời gian mưa ra Excel.

## Chức năng

- Đọc nhiều file TXT
- Khai thác dữ liệu từ DAILY_DATA → RAW_HOURLY_GRAPH_DATA
- Tự động lấy chuỗi mt, mr
- Xuất Excel

## Cài đặt

```bash
pip install -r requirements.txt
```

## Chạy chương trình

```bash
python main.py
```
